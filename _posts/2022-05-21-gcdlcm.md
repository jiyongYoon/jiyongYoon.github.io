# 최대공약수와 최소공배수

## 1) 최대공약수 (Greatest Common Denominator)

: 

### [약수]

:해당 수를 나누었을때 나머지가 없는 수.

8 -> 1, 2, 4, 8

12 -> 1, 2, 3, 4, 6, 12

**약수 중 가장 큰 수는 해당 수를 2로 나눈 값이기 때문에 for문을 돌 때는 끝까지 다 돌 필요 없음*

### [공약수]

: 두 수의 공통된 약수.

8과 12의 공약수 -> 1, 2, 4

### [최대공약수]

: 공약수 중 가장 큰 수.

8과 12의 최대공약수 -> 4

**for문을 돌아도 되나, 나중에 재귀함수로 잘 구할수 있음*



## 2) 최소공배수 (Lowest Common Multiple)

### [배수]

:해당 수를 2, 3, 4... 순차적으로 배한 수.

8 -> 8, 16, 24, ...

12 -> 12, 24, 36, ...

**약수 중 가장 큰 수는 해당 수를 2로 나눈 값이기 때문에 for문을 돌 때는 끝까지 다 돌 필요 없음*

### [공배수]

: 두 수의 공통된 배수.

8과 12의 공배수 -> 24, 48, ...

### [최소공배수]

: 공배수 중 가장 작은 수.

8과 12의 최소공배수 -> 24

**최소공배수 = 수1 X 수2 / 최대공약수*

**24 = 8 X 12 / 4*
